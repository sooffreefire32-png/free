# Lance-It
A professional freelancing marketplace connecting clients with freelancers.

## Getting Started
- Install Flutter from https://flutter.dev/
- Clone the repository.
- Run `flutter pub get`.
- Run the app using `flutter run` or build an APK with `flutter build apk`.

## Features
- Authentication (Login & Signup).
- Role selection: Client or Freelancer.
- Job Feed, Job Posting, Proposals.
- Profile Management & Messaging.

## Tech Stack
- Flutter (Frontend)
- Mock Backend Services
