# Lance-It Flutter Application

## Overview
Lance-It is a professional freelancing marketplace connecting clients and freelancers. This application is developed with Flutter 3.x and ensures scalability, clean architecture, and responsiveness.

## Features
- Authentication (Login/Signup).
- Role-based functionality (Clients and Freelancers).
- Job Posting and Bidding.
- Profile and Wallet Management.
- Messaging System.

## Installation
1. Install Flutter from [https://flutter.dev](https://flutter.dev).
2. Clone this repository.
3. Run `flutter pub get` to fetch dependencies.
4. Run `flutter build apk` to build the Android app.

## Additional Notes
Start with `lib/screens/auth/login_screen.dart` for the authentication flow.
