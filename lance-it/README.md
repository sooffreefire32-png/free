Lance-It: A Professional Freelancing Marketplace Application

## Overview
Lance-It connects clients with freelancers. Clients can post jobs, freelancers can bid on them, and approved freelancers can communicate via real-time chat and facilitate payments.

## Features
- Multi-role support (Clients/Freelancers)
- Real-time messaging
- Payment gateway
- Job posting and bidding
- Profile page and portfolio display

## Tech Stack
- **Frontend**: Flutter
- **Backend**: Firebase (Auth, Firestore, Cloud Functions, and Storage)

## Getting Started
1. Clone this repository.
2. Make sure you have Flutter SDK and Firebase CLI installed.
3. Run `flutter pub get` to get dependencies.
4. Set up Firebase project and configure Firebase rules.
5. Build the project using `flutter build apk`.
