# PROJECT: LANCE-IT - A Professional Freelancing Marketplace

## 1. CORE CONCEPT
Develop a full-stack-ready Flutter application named "Lance-It" that serves as a marketplace connecting "Clients" (who post jobs) and "Freelancers" (who bid on jobs). The app must be architected for scalability, using a clean directory structure and professional coding standards.

## 2. PROJECT STRUCTURE REQUIREMENTS
Generate a MASSIVE, complete Flutter project (at least 60+ files) including:
- **`lib/`**: Use Feature-first architecture.
- **`lib/features/auth/`**: (data, domain, presentation layers)
- **`lib/features/jobs/`**: (data, domain, presentation layers)
- **`lib/features/profile/`**: (data, domain, presentation layers)
- **`lib/features/chat/`**: (data, domain, presentation layers)
- **`lib/features/payments/`**: (data, domain, presentation layers)
- **`lib/core/`**: (theme, constants, errors, utils, network)
- **`lib/shared/`**: (widgets, models)
- **`android/`**: Full Android configuration including `gradle` files and `AndroidManifest.xml`.
- **`pubspec.yaml`**: Include all necessary dependencies.

## 3. KEY FEATURES
- Multi-role support (Client/Freelancer)
- Real-time chat
- Payment gateway UI
- Job posting and bidding flow
- Profile portfolios

**GOAL**: Produce a codebase that is ready for `flutter build apk` without manual code changes.
